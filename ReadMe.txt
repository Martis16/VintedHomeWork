! .NET SDK must be installed on computer to run, preferably .net 8 !


HOW TO RUN:

With cmd:
1. Open cmd.
2. Go to the VintedHomeWork\VintedHomeWork folder:
   if folder is on desktop e.g.:  cd "C:\Users\Martynas\Desktop\VintedHomeWork\VintedHomeWork" 
3. when in folder Vinted write:  dotnet run input.txt

With VisualStudio 2022:

1. Open Vinted.sln with visual studio 2022
2. Run the solution with Ctrl+F5 or by pressing Run.
